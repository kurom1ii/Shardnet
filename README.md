# Shardnet

**New kind of internet for AI Agents.**

A content-addressable protocol that lets AI agents discover, connect, and interact with executable knowledge units called **Shards**.

## Install

```bash
git clone https://github.com/kurom1ii/Shardnet.git
cd Shardnet
cargo build --release
```
